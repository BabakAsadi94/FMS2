# FMS2 Project Page

This folder contains the static project page for:

**FMS2: Unified Flow Matching for Segmentation and Synthesis of Thin Structures**

## Local Preview

Open `docs/index.html` directly in a browser.

## Publish With GitHub Pages

1. Push the repository to GitHub.
2. Open the repository settings.
3. Go to **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the branch, usually `main`.
6. Select the folder `/docs`.
7. Click **Save**.

GitHub will publish the page at a URL like:

`https://OWNER.github.io/REPOSITORY/`
